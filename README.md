# App setup on fresh macOS installation using Homebrew Bundle

## How to use

Install Homebrew:

`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

Download `Brewfile`, move to it's directory and run:

`brew bundle`
